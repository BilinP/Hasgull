Tokenizer.Tokenizer

Description

tokenize :: String -> Either String [Token] Source #

Tokenize the input string into a list of tokens.

This function removes comments and whitespace before invoking the tokenization of individual components.

stripWhiteSpace :: String -> [String] Source #

Strip whitespace from the input string and return a list of words.

Uses the built-in words function to split the input on whitespace.

removeComments :: String -> String Source #

Remove line comments from the input.

Cool gimmick we wanted to include in our compiler to be able add comments to our code. Line comments begin with "//" and continue until the end of the line.

removeLeadingWhiteSpace :: String -> String Source #

Remove leading whitespace from a string.

Drops any space characters (isSpace) from the beginning of the input.

tryReadIdentifierOrReservedWord :: String -> Token Source #

Determine if an identifier string corresponds to a reserved word, or should be treated as a generic identifier.

Recognized reserved words include Int, Void, "if", "else", etc. Any identifier not matching these is returned as an IdentifierToken.

tryReadMultiCharSymbol :: String -> Maybe (Token, String) Source #

Attempt to read a multi-character symbol from the input.

Recognized multi-character symbols include: * "=>" becomes ArrowToken * "==" becomes EqualsToken * "!=" becomes NotEqualToken

Returns a tuple of the matching token and the remaining input string, or Nothing if no match.

tryReadSymbolToken :: Char -> Maybe Token Source #

Attempt to read a symbol token from a single character.

Converts characters such as '=' and + into their corresponding token variants. Returns Nothing if the character is not a recognized symbol.

validLeftOver :: String -> Bool Source #

Determine if the remaining part of a numerical literal is valid.

A leftover is invalid if it starts with an alphabetic character, indicating an error.

tokenizationLoop :: String -> Either String [Token] Source #
